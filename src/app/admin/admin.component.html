Admin Dashboard Overview

Welcome to the Admin Dashboard! Here, you can manage and monitor employee leave records and summaries. As an admin, you can perform the following tasks:

  • Home: Return to the main dashboard to view all available options.
  • Leave Summary: Get an overview of employees who took leave today, this week, this month, and this year.
  • Month-wise Leave Summary: View leave data organized by month.
  • Week-wise Leave Summary: View leave data organized by week.
  • Date-wise Leave Summary: View leave data for a specific date.
  • Employee Leave Summary: Check the leave details for a specific employee.

Use the menu to navigate between these different sections and manage leave data efficiently.
